MapConductor iOS SDK ドキュメントへようこそ！

MapConductor は、複数のマッププロバイダーに共通の API を提供する統合マッピングライブラリです。このドキュメントでは、**v{CORE_MODULE_VERSION}** のパブリック API と統合ガイドを説明します。

- **[イントロダクション](/introduction/)** - MapConductor の機能概要
- **[はじめる](/get-started/)** - ステップバイステップのチュートリアル
- **[モジュール](/modules/)** - 利用可能なモジュールとコンポーネント

マッププロバイダーを選択してセットアップガイドに従ってください：

- **[Google Maps セットアップ](/setup/google-maps)** - Google Maps SDK の設定
- **[Mapbox セットアップ](/setup/mapbox)** - Mapbox の設定
