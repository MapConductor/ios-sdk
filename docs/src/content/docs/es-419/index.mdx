# Acerca de MapConductor
<MapConductorIntroduction
  text1="Al desarrollar una app de mapas en iOS, ¿alguna vez pensaste: 'La parte del mapa se siente demasiado complicada. Ojalá pudiera escribirla con el familiar SwiftUI...'? MapConductor hace realidad ese deseo. Con la API que provee MapConductor, puedes expresar fácilmente lo que quieres que haga el mapa mediante UI declarativa."
  text2="Se admiten múltiples SDKs de mapas: Google Maps, Mapbox, MapLibre, MapKit, ArcGIS, todos operados de la misma manera."
  text3="Incluso para funcionalidades que existen en un SDK de mapas pero no en otro, las hemos implementado nosotros mismos. Es decir, no solo agrupamos las APIs de cada SDK, sino que también construimos lo que falta."
  text4="Funcionalidades avanzadas como el agrupamiento de marcadores y los mapas de calor también pueden integrarse en pocas líneas."
  text5="¡Olvídate de tener que lidiar con una sección de mapas complicada en una app que por lo demás desarrollas con SwiftUI!"
/>
